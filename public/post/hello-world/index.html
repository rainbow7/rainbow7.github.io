你好，世界

这是我的第一篇博客文章，使用 Hugo + Even 主题搭建，部署在 GitHub Pages 上。

为什么选择 Hugo

  • 速度快：Go 语言编写，构建速度极快
  • 主题丰富：社区提供了大量优质主题
  • 部署简单：静态站点，配合 GitHub Actions 自动部署

写作流程

1
2

hugo new content post/my-new-post.md
hugo server -D

编辑 Markdown 文件，本地预览满意后推送到 GitHub，CI 自动构建部署。
